# Civic AI Platform - Ethiopia 🇪🇹

A multi-language complaint management system with AI analysis, real-time mapping, and government integration.

## Features
- 🎤 Voice input (English, Amharic, Oromo)
- 🗺️ Real-time heatmap visualization
- 🤖 AI text classification & sentiment analysis
- 📱 SMS/Email/Telegram notifications
- 🔐 Role-based access control
- 📊 Audit trail with cryptographic hashing
- 🔴 Fraud detection (duplicate complaints)
- ⚖️ Legal database integration
